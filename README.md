<h1>📋 Sistema de Cadastro com Banco de Dados</h1>

<h2>📖 Descrição</h2>

Este projeto consiste no desenvolvimento de um site de cadastro de usuários, com o objetivo de armazenar informações em um banco de dados de forma segura e organizada.

A aplicação permite o registro de dados por meio de uma interface web e utiliza integração com uma API para comunicação com o banco de dados. O Insomnia foi utilizado para testar e validar as requisições da API.
<p></p>
<h2>🎯 Objetivo</h2>

Desenvolver um sistema web capaz de:

• Realizar o cadastro de usuários 
<p></p>
• Armazenar dados em um banco de dados
<p></p>
• Testar requisições HTTP utilizando o Insomnia 
<p></p>
• Praticar conceitos de front-end e back-end
<p></p>
<h2>🛠️ Tecnologias Utilizadas</h2>

• HTML – estrutura do site 
<p></p>
• CSS – estilização da interface
<p></p>
• JavaScript – interatividade e lógica 
<p></p>
• Banco de Dados – armazenamento das informações 
<p></p>
• Insomnia – teste das requisições da API
<p></p>
<h2>⚙️ Funcionalidades</h2>

• Cadastro de usuários 
<p></p>
• Envio de dados para o servidor
<p></p>
• Armazenamento em banco de dados 
<p></p>
• Testes de API com Insomnia
<p></p>
